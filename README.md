# COVID-19 CT-Scan Classification

This repository contains my solution and experiments for a deep learning project based on the  
**SARS-CoV-2 CT-Scan Dataset** from Kaggle.

The goal is to classify chest CT images as **COVID-19** or **Non-COVID** using Convolutional Neural Networks (CNNs).
